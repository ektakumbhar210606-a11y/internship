1. Html Basic

Description: core fundamentals of HTML (HyperText Markup Language) and created simple webpages to understand how structure, formatting, and basic elements work.

Key Features Implemented:

  • Basic HTML document structure
  • Understanding elements, tags, attributes, and nesting
  • Working with headings and paragraphs
  • Adding text formatting like bold, italic, underline
  • Creating hyperlinks using
  • Introduction to semantic HTML tags
  • Understanding block-level vs inline elements

2. CSS Basics

Description: Learned the core fundamentals of CSS (Cascading Style Sheets) and practiced applying styles to HTML elements to improve visual appearance, layout, and user interface design.

Content Covered:

  • Understanding CSS syntax: selectors, properties, and values
  • Applied styles using inline, internal, and external CSS
  • Styled text using font size, color, weight, and family
  • Worked with the CSS Box Model (margin, padding, border)
  • Used class and ID selectors for targeted styling
  • Applied background colors, borders, and spacing to elements
  • Introduction to layout techniques using Flexbox

3. Basic Demonstration Projects

Project Description: A series of smaller demonstration projects created to reinforce learning and explore various CSS patterns and techniques.

Content Covered:

  • Multiple centering techniques (Flexbox, Grid, positioning)
  • CSS Grid basic demonstrations
  • Layout patterns and responsive behaviors
  • Practical implementation of common design patterns

4. Form Implementation (Form.html)

Project Description: Created a comprehensive HTML form demonstrating various form elements, input types, and validation techniques commonly used in web applications.

Key Components:

  • Personal details fieldset with legend
  • Text input fields (First Name, Last Name, Email)
  • Radio buttons for single-choice selection (Gender)
  • Checkboxes for multiple-choice options (Hobbies)
  • Dropdown select menus (Car selection)
  • Form validation attributes (required, pattern, type)
  • Submit and reset buttons

5. Image Gallery

Project Description: Designed and implemented a responsive image gallery showcasing proper image management, layout techniques, and interactive hover effects.

Key Features:

  • Flexbox-based responsive grid layout
  • Multiple gallery items with consistent sizing
  • Image descriptions and captions
  • Smooth hover effects with scale transformations
  • Responsive behavior adapting to viewport size
  • Visual feedback on user interaction
