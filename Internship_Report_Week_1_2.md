## WEEK 1: FOUNDATIONS & HTML BASICS

### Learning Objectives
- Understand HTML structure and semantic markup
- Learn CSS fundamentals for styling and layout
- Explore responsive design principles
- Implement basic HTML forms and galleries

### Projects Completed

#### 1. **Calculator Application** (`Calc.html`)
- **Description:** A functional calculator interface built using HTML and CSS
- **Key Features:**
  - Display screen for input and output
  - Grid-based button layout for numeric and operation buttons
  - Responsive design
  - Clean and intuitive user interface
- **Technologies Used:** HTML5, CSS3
- **Learning Outcomes:** 
  - Understanding of flexbox layout
  - DOM structure for interactive elements
  - CSS styling for calculator components

#### 2. **Form Implementation** (`Form.html`)
- **Description:** A comprehensive form demonstrating various form elements and input types
- **Key Components:**
  - Personal details fieldset
  - Text input fields (First Name, Last Name)
  - Radio buttons (Gender selection)
  - Checkboxes (Hobbies selection)
  - Dropdown menus (Car selection)
  - Form validation attributes
- **Technologies Used:** HTML5 Form Elements
- **Learning Outcomes:**
  - Form element implementation
  - Input validation basics
  - User interaction patterns
  - Accessibility considerations

#### 3. **Image Gallery** (`Gallery.html`)
- **Description:** A flexible gallery layout showcasing image management
- **Key Features:**
  - Flexbox-based responsive grid layout
  - Hover effects on gallery items
  - Image descriptions and captions
  - Smooth transitions and visual feedback
- **Technologies Used:** HTML5, CSS3 (Flexbox)
- **Learning Outcomes:**
  - Understanding of CSS Grid and Flexbox
  - Responsive image handling
  - Hover effects and transitions
  - Gallery pattern implementation

#### 4. **Positioning & Layout** (`Position.html`, `Position2.html`)
- **Description:** Deep dive into CSS positioning techniques
- **Key Concepts Explored:**
  - Static positioning (default)
  - Relative positioning
  - Absolute positioning
  - Fixed positioning
  - Z-index stacking
- **Technologies Used:** HTML5, CSS3 Positioning
- **Learning Outcomes:**
  - Understanding of CSS positioning models
  - Practical application of positioning in layouts
  - Stacking context and z-index management

#### 5. **Demo Projects** (`Demo.html`, `Center.html`, `Sample.html`)
- **Description:** Various demonstration projects to reinforce learning
- **Content:**
  - Centered layouts
  - CSS grid demonstrations
  - Layout patterns and best practices
- **Learning Outcomes:**
  - Practical implementation of various CSS patterns
  - Center alignment techniques (flexbox, grid, etc.)
